The files in this directory contain the URL's of the test data.

i.e. text_urls.txt contains the sample text data

The naming scheme within each of this file is:

"NAME OF THE FILE"
"URL"

Examine the file for understanding.

There is also an additional file which contains a brief description of
the data

i.e. text_urls_info.txt

NOTE:
Some of the files may be compressed as tar.gz or .zip. Make sure that
you decompress them before further use.
